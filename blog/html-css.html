HTML och CSS - Datum

Vårt äventyr som Webutvecklare inom .Net började i Augusti 2021. Här träffade vi vår lärare Björn och insåg snart att vi var många som hade samma ide. Detta skulle bli ett äventyr i sig att komma igång med studier, att det var mycket att lära sig för att komma igång med att vara utvecklare.

Vi fick snart börja lära oss C#, Github, men också hur viktigt det är i vår branch att nätverka. Så vi hade tidigt en föreläsning om bland annat Linked in och vi ombeddes att lägga till varandra. Det kan ju hända att många av oss är kollegor i framtiden.
